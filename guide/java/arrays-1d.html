1D Arrays

An array is a collection of values of the same type (e.g. a collection of ints). Java Arrays are immutable, which means that the array's length is fixed. You can not add more values to the end of an array, you can only modify existing ones. Arrays are zero-indexed, this means that the first item in the array is at index 0.

Declaration

To declare an array, it is very similar to a variable. There are multiple valid ways to declare an array.
They can be seen below:

type[] name; // Similar to int x;
type name[]; // Similar to int x;
type[] name = new type[length];
type name[] = new type[length];

You can also declare an array and assign it values at the same time:

int[] arr1 = {1, 2}; // int array with length 2 and the values of 1 and 2
int[] arr2 = new int[]{1, 2}; // int array with length 2 and the values of 1 and 2

int arr3[] = {1, 2, 3, 4}; // int array with length 4 and the values of 1, 2, 3, 4
int arr4[] = new int[]{1, 2}; // int array with length 2 and the values of 1 and 2

Modifying Array Values

You can modify any value of a Java array but not add more values to the array To change the value at a specific index use this syntax:

name[index] = value;

Let's use it, shall we?

String[] names = new String[3]; // Valid indices are 0, 1, and 2
names[0] = “John”;
names[1] = “Nancy”;
names[2] = “Jake”;
names[3] = “Teresa” // This will result in an OutOfBound exception as we are attempting to modify item #4 of an array with a length of 3

You can also changed already initialized values.

double[] nums = {1.1, 2.1, 3.3};
nums[1] = 2.2; // Changes the item #2 of the array nums from 2.1 to 2.2

You can also get the value of a specific index of an array, with the same syntax

double[] nums = {1.1, 2.1, 3.3};
System.out.println(nums[0]); // Outputs: 1.1

You can NOT print the contents of an array like this (as you may expect):

int[] arr1 = {1, 2};
System.out.println(arr1);

This code will instead print some random string as arrays are objects (which are reference types, just like Strings), so printing an array will print its memory location (which looks something like this: [I@15db9742) and not its contents. You can print the contents of an array by using for loops.

Array Length

To get the length of an array, use the following syntax:

name.length;

It is important to realize that since the array is 0-indexed, there is no element at name[name.length], instead the last element in an array can be accessed with name[name.length - 1].
