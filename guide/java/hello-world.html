One thing to note is that Java doesn't technically have functions. Instead, there is something called a method, which actually belongs to an object (or to a class which we'll later discuss).
Console: The output of a program
System.out.println(): The method used to print a line to the console
To output to the console use System.out.println(what to print);

// Outputs “Hello World” to the console
System.out.println(“Hello World”);

You can use variables to print variables to the console like this:

String name = “John”;
// Outputs “John” to the console
System.out.println(name);
